== Description ==
* Easily add the Pirobox Extended lightbox to all images to your blog.
* Create links to embeded videos with ease.
* All options can be set globally
* Options can be overriden on a per page basis
* Supports custom post types

== Installation ==

For an automatic installation through WordPress:

1. Go to the 'Add New' plugins screen in your WordPress admin area
2. Search for 'Pirobox Extended'
3. Click 'Install Now' and activate the plugin
4. Go to Settings > Pirobox Extended to edit all global options
5. You can setup Pirobox Options also in post edit-page  and add Videos from the button above the textarea

For a manual installation via FTP:

1. Upload `the pirobox_wp` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' screen in your WordPress admin area

To upload the plugin through WordPress, instead of FTP:

1. Upload the downloaded zip file on the 'Add New' plugins screen (see the 'Upload' tab) in your WordPress admin area and activate.

Note: due to confusion, there are no longer separate versions for PHP4 and PHP5.
